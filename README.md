The Phone Number Information Tracker is a simple Python project that extracts useful details from a phone number, such as country/region, telecom carrier, and associated time zones.
It uses the phonenumbers and pycountry libraries to parse and analyze international phone numbers accurately.

This project demonstrates Python scripting, external library usage, and basic data extraction, making it suitable for beginners and data-oriented applications.
